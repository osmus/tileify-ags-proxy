ArcGIS Server Proxy

Use ArcGIS Server map services as background layers in OpenStreetMap editors like iD and JOSM.

Paste your ArcGIS Server map service URL (ending with ..../MapServer ) below.

Use the URL parameters to specify service properties, like:

transparent | true (transparent background)
layers | show:14 (shows layer 14 in the service)

For all options see Esri Rest Service Docs.

Click Update Tile Layer to refresh the map.
